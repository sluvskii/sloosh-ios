# NeoMovies теперь в Telegram

У NeoMovies появился собственный Telegram-бот для просмотра фильмов, сериалов и аниме через встроенные плееры и напрямую внутри Telegram.

## Что доступно

- быстрый доступ к каталогу без отдельного приложения;
- просмотр через Telegram-плеер или через внешнюю выдачу;
- отдельная коллекция, которая будет постепенно расширяться.
