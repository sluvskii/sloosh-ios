# Neo ID вынесен в отдельный сервис

Для экосистемы NeoMovies появился отдельный сервис авторизации `Neo ID`. Через него теперь строится единая авторизация, а сама инфраструктура стала чище и прозрачнее.

## Что изменилось

- сервис авторизации вынесен отдельно в `Neo ID`;
- документация API перенесена на отдельный адрес;
- появилась более явная структура между сайтом, API и системой идентификации.
