# NeoMovies Android v0.1.0-pre1

Первый пререлиз мобильного клиента NeoMovies. На этом этапе появилась базовая Android-сборка, стартовая структура проекта и первые публичные материалы для тестирования.

## Что вошло в первую сборку

- базовый мобильный клиент для просмотра контента;
- стартовая навигация по приложению;
- первые публичные сборки для ручного тестирования.

## Что это значило для проекта

С `v0.1.0-pre1` проект вышел из внутренней стадии и получил первую публичную Android-сборку, от которой дальше уже росли TV-ветка, TorrServer и улучшения интерфейса.
