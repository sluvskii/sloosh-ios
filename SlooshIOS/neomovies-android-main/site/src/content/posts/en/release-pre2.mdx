# NeoMovies Android v0.1.0-pre2

The second prerelease made the project significantly more practical: `Collaps` was added as a source, and a dedicated TV branch with early Android TV builds appeared.

## Main changes

- added the `Collaps` source;
- introduced a separate TV UI branch;
- added TV build workflows and first TV packages;
- brought up the initial project website infrastructure in parallel.

## Why it mattered

`v0.1.0-pre2` was the point where NeoMovies stopped being only a mobile experiment and started moving toward a unified phone-and-TV product.
