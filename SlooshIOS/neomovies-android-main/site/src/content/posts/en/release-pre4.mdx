# NeoMovies Android v0.1.0-pre4

`v0.1.0-pre4` delivered the most substantial playback-related update so far: the TV player moved much closer to a working daily build, the details screen was cleaned up, and watch progress handling improved.

## Main changes

- fixed `auth` and the network layer for Neo ID integration;
- improved watch progress display;
- fixed resume behavior on the details screen;
- improved the TV audio track selector;
- removed hardcoded strings from the details screen;
- refined `TvWatchSelectorScreen`, `PlayerViewModel`, and TV seek/indicator components.

## Result

This prerelease feels much closer to a practical testing build: fewer visual issues, better player behavior, and a clearer resume flow.
