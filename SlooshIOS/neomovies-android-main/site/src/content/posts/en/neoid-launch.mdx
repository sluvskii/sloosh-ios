# Neo ID moved to a dedicated service

The NeoMovies ecosystem now includes a dedicated `Neo ID` authentication service. This makes the authorization flow cleaner and separates the website, API, and identity system more clearly.

## What changed

- the authentication service was moved into `Neo ID`;
- API documentation was moved to a dedicated address;
- the ecosystem structure became clearer between the website, API, and identity layer.

- Website: [www.neomovies.ru](https://www.neomovies.ru)
- API docs: [api.neomovies.ru](https://api.neomovies.ru)
- Neo ID: [id.neomovies.ru](https://id.neomovies.ru)
